# TranscriptHub

A professional YouTube transcript downloader with a clean GUI interface. Download video captions/subtitles and save them as text or markdown files.

## Features

- Simple and intuitive GUI interface
- Download transcripts from any YouTube video with captions
- Save transcripts as plain text (.txt) or Markdown (.md)
- Preview transcripts before saving
- Automatic video title detection
- Smart formatting for better readability

## Requirements

- Python 3.6+
- tkinter (usually comes with Python)

## Installation

Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the program:
```bash
python3 TranscriptHub.py
```

1. Copy any YouTube URL that has captions/subtitles enabled
2. Paste it into the URL field
3. Click "Get Transcript"
4. Choose your preferred format (Text or Markdown)
5. Click "Save to File" to save the transcript

## Note

This tool only works with YouTube videos that have captions/subtitles enabled. Most official channels, educational content, and popular videos have captions available.
